## Overview
**AssignmentBuddy** is a **microservices-based backend system** for secure PDF upload, text extraction, and AI-driven question answering. The system demonstrates modern backend architecture using Spring Boot, Spring Cloud, JWT, and MongoDB, making it scalable, secure, and maintainable.
